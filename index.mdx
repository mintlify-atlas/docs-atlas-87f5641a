    </p>

  <Steps>
    <Step title="Set up your account">
      Sign up for the IGAD Innovation Hub and configure your AWS credentials. The platform uses AWS Cognito for secure authentication.

      ```bash
      # Set your AWS profile
      export AWS_PROFILE=IBD-DEV
      export AWS_REGION=us-east-1
      ```
    </Step>

    <Step title="Upload your RFP document">
      Upload the Request for Proposal (RFP) document you need to respond to. The system supports PDF, DOCX, and other common formats.

      <Note>
        The platform uses AWS Bedrock with Claude for intelligent document analysis, extracting key requirements, evaluation criteria, and submission guidelines automatically.
      </Note>
    </Step>

    <Step title="Add your concept materials">
      Provide your initial concept document or reference proposals. The AI will analyze these alongside the RFP to generate aligned content.

      ```json Example concept structure
      {
        "title": "Climate-Smart Agriculture Initiative",
        "objectives": ["Improve crop resilience", "Reduce water usage"],
        "methodology": "Participatory action research"
      }
      ```
    </Step>

    <Step title="Generate and refine">
      Let the AI analyze your documents, generate a concept document, create a proposal structure, and provide draft feedback. Review each step and refine as needed.

      <Accordion title="Example AI-generated output">
        The system produces structured sections including:
        - Executive Summary aligned with RFP requirements
        - Technical Approach matching evaluation criteria
        - Budget framework based on RFP guidelines
        - Risk mitigation strategies
      </Accordion>
    </Step>
  </Steps>
</div>
